·+ DESPAISAJES (De-landscapes)
/ IDENTITY /
/ MOTION /
2024

ENG

Through an exhibition, this project explores "de-landscapes," contemporary landscapes altered by suburban expansions, industrial installations, infrastructures...A visual identity was developed that symbolizes the fragmentation and exploitation of the landscape. Additionally, an interactive audiovisual installation was created with TouchDesigner, using real-time motion detection: when the viewer moves, the video degrades; if they remain still, the landscape is preserved. This symbolizes the impact of human activities. The exhibition is complemented by a collection of locations illustrating de-landscapes.

ESP

Este proyecto plantea una exposición que explora los "despaisajes", paisajes contemporáneos alterados por expansiones suburbanas, instalaciones industriales, infraestructuras... Se desarrolló una identidad visual que simboliza la fragmentación y explotación del paisaje. Además, se creó una instalación audiovisual interactiva con TouchDesigner, que utiliza detección de movimiento en tiempo real: cuando el espectador se mueve, el vídeo se degrada; si permanece inmóvil, el paisaje se conserva. Esto simboliza el impacto de las actividades humanas. La exposición se complementa con una colección de localizaciones que ilustran despaisajes.
